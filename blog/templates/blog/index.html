Django

Django és un framework popular de desenvolupament web de codi obert que s'utilitza per crear aplicacions web robustes i escalables en Python. Va ser creat l'any 2003 per Adrian Holovaty i Simon Willison mentre treballaven en el lloc web de notícies Lawrence Journal-World.

Django segueix el patró de disseny Model-Vista-Controlador (MVC), que separa la lògica de presentació de les dades i la interacció amb l'usuari. El framework també proporciona eines pel treball amb bases de dades, l'autenticació d'usuaris i la gestió de formularis, entre altres característiques.
